1.     Function

To synchronize files between a PC/NAS and Android device. Three Synchronization methods are available.（mirror, move and copies）

         Mirror

If master and target files are different, to copied files from master to the target directory. To delete target files and directories if does not exist in the master.

         Move

If master and target files are different, to copied files from master to the target directory. To delete the master directories and files copied to the target.

However, the master and target same name, same file modification time and file size, to delete the file on the master without copying files.

         Copy

If master and target files are different, to copied files from master to the target directory. However, the same file modification time and file size with the same name does not copy the master and target.

If the determination is satisfied and the differential file one of the following three conditions.

         File does not exist

         Different file sizes

         Different over when last updated 3 seconds

2.     How to use

If you have the wrong setting, move or mirror will delete the file, and copy will override the existing file. Please make sure to use a test folder for test run.

         Local profile is specifies the local profile directory of Android.

         Remote profiles is specifies the directory server, such as the SMB NAS/PC.

         Synchronization profile is defined a Remote/Local profile to the master/target.

Synchronization is performed for each synchronization profile.

         Determine the difference in only the last modification time

         Determine the difference in time and last update time + time zone last updated

         Do not use the last modified file time

If the file’s last modification time of the media file store folder by restarting the terminal rewritten to be copied each time, please try the (Do not use the last modified file time).

5.2 Sample code

The following code sample to start SMBSync from Activity.

 

public class MainActivity extends Activity {

@Override

protected void onCreate(Bundle savedInstanceState) {

super.onCreate(savedInstanceState);

setContentView(R.layout.activity_main);

}

@Override

protected void onResume() {

super.onResume();

 

PackageManager pm = getPackageManager();

Intent in=pm.getLaunchIntentForPackage("com.sentaroh.android.SMBSync");

in.setAction(Intent.ACTION_MAIN);

in.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);

in.putExtra("AutoStart", true);

in.putExtra("AutoTerm", true);

in.putExtra("Background", true);

String[] prof=new String[] {"S-PGMS","S-Pictures"};

in.putExtra("SyncProfile", prof);

startActivity(in);

 

}

}
